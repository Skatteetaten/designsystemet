## Hvordan bruke padding

Padding kan settes på to måter:

1. Hvis **padding**-prop er brukt så er denne verdien lik for alle brekkpunkter.
2. Hvis **hasResponsivePadding** = true OG valgfritt egendefinert stilsett satt i **classNames** så har man mulighet for å definere forskjellig padding for hvert av <Link url="/?path=/docs/designtokens-breakpoints--docs">Desginsystemet brekkpunkter</Link>. Stilene satt i **classNames.padding** er mobile-first, dvs. at variablene settes fra minste bredde til største (min-width). Default-verdiene ved bruk av **hasResponsivePadding** er --spacing-m i området mellom 0px og --breakpoint-xs. --spacing-xl gjelder fra --breakpoint-xs til alt som er bredere.

Bruk av **padding** utelukker bruk av **hasResponsivePadding** og omvendt.

### Eksempel

```css
// Egen CSS i fil Panel.scss ELLER definert globalt i prosjektet hvis man ønsker samme padding på alle Panel'er
.panel {
  // Bestemmer at padding --spacing-xl skal gjelde for alle skjermbredder bredere enn --breakpoint-xs
  --panel-padding-xs: var(--spacing-xl);
  // Bestemmer at padding 30px skal gjelde for alle skjermbredder bredere enn --breakpoint-s
  --panel-padding-s: 30px;
  // Bestemmer at padding --spacing-m skal gjelde for alle skjermbredder bredere enn --breakpoint-m
  --panel-padding-m: var(--spacing-m);
}
/* Følgende tre CSS-variabler kan også settes av konsumentkoden
--panel-padding-mobile
--panel-padding-l
--panel-padding-xl
*/
```

```tsx
// Egen komponent
import { Panel } from '@skatteetaten/ds-content';
import styles from './Panel.scss';
<Panel
  title={`Med responsive padding`}
  classNames={styles.panel}
  hasResponsivePadding
>
  <Paragraph>Innhold i Panel</Paragraph>
</Panel>;
```
