# Kommandolinjeverktøy

`@skatteetaten/ds-cli` er et kommandolinjeverktøy for å håndtere kode-transformasjoner og andre oppgaver relatert til designsystemet. Dette verktøyet hjelper utviklere med å oppdatere og vedlikeholde kodebasen ved å tilby automatiserte transformasjoner for breaking changes.

## Kommandoer

### Migrate

Bruk `migrate`-kommandoen for å utføre kode-transformasjoner.

```bash
npx @skatteetaten/ds-cli migrate <migration_name> <path>
```

#### Alternativer:

- `--dry, -d`: Forhåndsvis endringer uten å utføre dem.
- `--exclude, -e <pattern>`: Ekskluder filer/mapper (kan brukes flere ganger).
- `--list, -l`: Vis tilgjengelige migreringer.
- `--help, -h`: Vis hjelp for kommandoen.

#### Eksempler:

```bash
# Vis tilgjengelige migreringer
npx @skatteetaten/ds-cli migrate --list

# Utfør migrering
npx @skatteetaten/ds-cli migrate v2-alert src/

# Forhåndsvis endringer uten å utføre dem
npx @skatteetaten/ds-cli migrate v2-textarea src/ --dry

# Ekskluder spesifikke filer/mapper
npx @skatteetaten/ds-cli migrate v2-alert src/ --exclude "*.test.js" --exclude node_modules
```

### Versjon

For å sjekke hvilken versjon av `ds-cli` du har installert, bruk `version`-kommandoen eller `--version`:

```bash
npx @skatteetaten/ds-cli version
npx @skatteetaten/ds-cli --version
```

### Hjelp

For å få en oversikt over tilgjengelige kommandoer og alternativer:

```bash
npx @skatteetaten/ds-cli --help
npx @skatteetaten/ds-cli migrate --help
```
